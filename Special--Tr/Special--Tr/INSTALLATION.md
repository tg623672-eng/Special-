# Welcome to SK Host!

SK Host is a theme for Pterodactyl built with Blueprint, providing a smooth
installation experience and great compatibility with other extensions.

## Extension installation

Before you can start using SK Host, you'll need Blueprint. Blueprint allows themes
like SK Host to have great compatibility with other modifications made by
extensions. You can find more information, download links and documentation at
**https://blueprint.zip**.

After you've installed Blueprint, drag and drop the `.blueprint` file over to your
Pterodactyl folder (commonly /var/www/pterodactyl) and run `blueprint -install nebula`.

You can find SK Host's configuration options in your admin panel
(admin > extensions > SK Host).

## Configuration

To make SK Host fully yours, explore **SK Host Designer**, which stores all
configuration options for the theme and gives you an easy interface to configure
everything in.
